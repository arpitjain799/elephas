BaseParameterServer

elephas.parameter.server.BaseParameterServer(model: keras.engine.training.Model, port: int, mode: str, **kwargs)

BaseParameterServer

Parameter servers can be started and stopped. Server implementations have to cater to the needs of their respective BaseParameterClient instances.

[source]

HttpServer

elephas.parameter.server.HttpServer(model: keras.engine.training.Model, port: int, mode: str, **kwargs)

HttpServer

Flask HTTP server. Defines two routes, /parameters to GET current parameters held by this server, and /update which can be used to POST updates.
